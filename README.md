## Sprint_1

Приложение на Netlify: https://glittery-chimera-e5a6f6.netlify.app/


<h3 align="center">Запуск:</h3>

- запуск проекта в dev-режиме:

```bash
npm run dev
```

- запуск проекта:

```bash
npm run start
``` 

- запуск сборки проекта:

```bash
npm run build
```

`http://localhost:3000/` - URL localhost
